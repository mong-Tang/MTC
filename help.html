빠른 시작

Quick Start

  • 파일 > 파일 열기로 ZIP/CBZ를 선택합니다.
  • CBZ는 변환 후 열기 여부를 물어봅니다. 아니오를 누르면 그대로 엽니다.
  • 보기 > 이동에서 페이지 이동/권 이동을 사용할 수 있습니다.
  • Open a ZIP/CBZ from File > Open File.
  • For CBZ, the app asks whether to convert first. Choose No to open as-is.
  • Use View > Navigate for page/book navigation.

단축키

Shortcuts

  • 이전/다음 페이지: ← / →
  • 10페이지 이동: PageUp / PageDown
  • 처음/마지막 페이지: Home / End
  • 이전권/다음권: Shift+PageUp / Shift+PageDown
  • 목록 패널 토글: Ctrl+B
  • Previous/Next Page: Left / Right
  • Jump 10 pages: PageUp / PageDown
  • First/Last Page: Home / End
  • Previous/Next Book: Shift+PageUp / Shift+PageDown
  • Toggle List Panel: Ctrl+B

페이지 편집

Page Editing

  • 편집은 원본을 직접 덮어쓰지 않습니다.
  • 원본 ZIP이면 _edited_YYYYMMDD_HHMMSS.zip 새 파일을 만듭니다.
  • 이미 편집본이면 같은 파일을 갱신하고 즉시 리로드합니다.
  • Editing never overwrites original source ZIP directly.
  • For original ZIPs, a new _edited_YYYYMMDD_HHMMSS.zip is created.
  • For edited ZIPs, the same file is updated and reloaded immediately.

CBZ 처리 방식

CBZ Handling

  • 앱은 ZIP을 우선 권장하며, CBZ 열기 시 변환 여부를 확인합니다.
  • CBZ가 유효한 ZIP 구조면 변환 없이도 바로 열 수 있습니다.
  • 가짜 확장자/손상 파일은 검사 단계에서 안내 메시지를 확인하세요.
  • The app prefers ZIP and asks before converting CBZ.
  • If CBZ is a valid ZIP container, it can be opened without conversion.
  • For fake extensions or corrupted files, check the guidance message shown during validation.

문제 해결

Troubleshooting

  • ZIP(CBZ) 형식 아님: 확장자만 바뀐 파일일 수 있습니다.
  • 손상: 파일을 다시 다운로드/복사해 보세요.
  • 권한 없음: 다른 앱 점유 여부 또는 폴더 권한을 확인하세요.
  • Not a valid ZIP(CBZ): The file may only be renamed.
  • Corrupted: Re-download or re-copy the file.
  • Access denied: Check file lock and folder permissions.
