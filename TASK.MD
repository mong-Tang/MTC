* [ ] 🗺️ **전체 다이어그램 설계**: 단순 단일 변환 -> 다중 파일 입출력 복합 흐름도로 진화
* [ ] 📋 **UI 확장 설계**: 다중 파일 드래그앤드롭 목록 및 '묶기/나누기' 모드 전환 스위치 배치

## ⚡ 개발 구현 과제 (Implementation)

### 1️⃣ 병합 엔진 (Archive Merger)
* [ ] 🔗 **여러 권 묶기**: 다수의 ZIP/7z 파일을 순서대로 병합하여 단일 대용량 파일로 압축 생성
* [ ] 🔢 **순서 보장 로직**: 파일명(01, 02...)을 인식하여 내부 페이지 정렬 순서 무결성 확보

### 2️⃣ 분할 엔진 (Archive Splitter)
* [ ] ✂️ **대용량 쪼개기**: 너무 큰 단일 파일을 지정된 기준(페이지 수 or 용량)으로 자동 분절
* [ ] 🏷️ **네이밍 자동화**: 쪼개진 파일에 `_Part1`, `_Part2` 또는 순번 자동 부여 정책 구현

### 3️⃣ UI/UX 고도화
* [ ] 📥 **멀티 셀렉터**: 단일 파일 선택기 -> 복수 파일 리스트 뷰어로 입력 영역 업그레이드
* [ ] 📊 **동적 프로그레스**: 전체 병합/분할 진행률을 시각화하는 멀티 스테이지 프로그레스 바 도입
